对比度敏感函数在HDR编码中的应用

摘要：随着超高清电视（UHDTV）时代的到来，高动态范围（HDR）成像技术成为提供沉浸式视觉体验的关键推动力。本文深入探讨对比度敏感函数（Contrast Sensitivity Function, CSF）在HDR编码中的核心应用。从经典的亮度感知模型（Weber-Fechner定律、Stevens幂律）出发，系统阐述了Gamma校正、HLG曲线的基本原理，重点推导了Barten CSF模型的物理基础及其在感知量化器（Perceptual Quantizer, PQ）设计中的关键作用，并介绍了PQ编码的验证实验。本文旨在为理解HDR编码曲线的视觉心理学基础提供一份清晰的导论。

一、引言

随着我们步入超高清电视（UHDTV）的新纪元，人类对"真实"与"沉浸"视觉体验的追求达到了前所未有的高度。在众多突破性技术中，高动态范围（High Dynamic Range, HDR）成像脱颖而出，成为UHDTV卓越品质的关键使能技术[1][2]。

HDR成像标志着从标准动态范围（SDR）成像的重大飞跃——通过同时丰富图像亮部和暗部的细节，HDR为终端用户提供了视觉上更丰富、更逼真、更引人入胜的观赏体验。为了高效运作HDR成像并防止可见伪影的引入，需要一个强大的亮度感知模型[3]——这正是编码曲线发挥关键作用的领域。

国际电信联盟（ITU）推动了两条独特HDR编码曲线的发展：感知量化器（Perceptual Quantizer, PQ，由电光传递函数EOTF定义）和混合对数伽马（Hybrid Log-Gamma, HLG）。本文将重点阐述PQ的概念与推导过程，以及其背后的核心视觉科学原理——对比度敏感函数（CSF）。

二、亮度感知的对数与幂律视角

在深入研究PQ编码和CSF之前，需要理解这些变换的起源及其背后的基本视觉理论。两个关键原则阐明了亮度感知的本质[4]：Weber-Fechner定律和Stevens定律。

2.1 Weber-Fechner 定律：对数视角

对于重量为I的物体，人能够分辨的最小重量差ΔI满足：ΔI / I = k。Fechner在此基础上推导出物理刺激与心理反应的对数关系：S = a + k' log(I)。这一发现成为显示系统Gamma校正的理论基石。

2.2 Stevens 幂律

S. S. Stevens通过大量心理物理学实验[5]提出幂律方程：S = k · Iα。其中α = 0.5对应DeVries-Rose定律，与人眼对较暗刺激的感知更为吻合。

2.3 ITU.709 中的 Gamma 校正

ITU.709标准中Gamma OETF具有指数α = 0.45，与DeVries-Rose亮度感知定律展现出显著的一致性。然而Gamma校正基于50~100 cd/m²的有限动态范围（小于3个对数单位），随着显示亮度和动态范围的指数增长，Gamma校正面临维持其性能的挑战。

2.4 混合对数伽马（HLG）

图2：JND作为背景亮度的函数（来源：Barlow, 1965）

HLG的成功在于它结合了经典亮度感知模型的精华。其设计根据亮度水平调整行为：高亮度值遵循Weber-Fechner对数定律，保持物理与心理响应的对数关系；低亮度值遵循DeVries-Rose定律，更准确地共振于人类对暗刺激的感知。这种独特设计使HLG不仅克服了Gamma校正的不足，还增强了设备兼容性。

三、对比度敏感函数（CSF）

图3：人眼敏感度随空间频率的变化（CSF曲线）

在视觉感知与成像领域，经典亮度感知模型做出了重要贡献。然而，这些模型传统上未能考虑空间频率的影响。空间频率是图像中亮度变化的次数，在同一图像的不同区域可有显著差异。

图4：图像包含不同空间频率的区域（照片：Alan/宋思远）

Campbell和Robson引入了对比度敏感函数（Contrast Sensitivity Function, CSF）。这一革命性模型带来了范式转变——提出刺激的空间频率影响我们检测对比度变化的能力。因此，将CSF纳入成像模型的编解码过程是合乎逻辑的。

3.1 Barten 的 CSF 模型

Barten引入了一个基于物理与光学原理、并通过大量实验数据拟合的CSF模型[6]。该模型以对比度敏感度（S）——衡量我们在不同空间频率下辨别亮度对比变化的内在能力——和亮度（L, cd/m²）作为关键输入。

Barten的CSF模型通过考虑视觉系统内部固有产生的内部噪声来推导。其噪声积分形式的起点为：

mt = k mn

通过考虑光学系统的调制传递函数（MTF）和侧抑制效应，模型增强为：

mt Mopt(u) Mlat(u) = k mn

通过傅里叶分析，噪声表达为 mn = 2√(Φn / XYT)，其中X, Y, T分别为噪声覆盖的空间和时间维度。

噪声形成为：Φn = Φph M2lat(u) + Φ0

其中Φph 表示每单位时间吸收的光子数，Φ0 为与光水平无关的内部噪声。

最终获得Barten CSF的完整形式——整合了视觉系统内部噪声、光学系统MTF以及侧抑制等多个方面的全面人类视觉感知表示。

图5-7：Barten CSF模型的完整推导过程及最终公式

四、运用Barten CSF模型优化感知EOTF

ITU已在BT.2246建议书中采用了Barten CSF的缩放版本，有效地展示了BT.1886建议书下HDTV的EOTF如何反映人类感知行为。

Miller及其同事[7]更进一步，提出了直接应用Barten CSF来计算优化的感知EOTF——这就是PQ曲线的理论基础。该方法确保每个亮度间隔精确对应于Barten CSF的恰可察觉差（Just Noticeable Difference, JND），包括两个关键参数：标准视角X0 = 40°和跟踪视觉系统最大敏感度的空间频率。

4.1 PQ 的推导

确定了核心参数后，可以建立JND。JND方程涉及逐项亮度，表达为：

mt = (Lj+1 − Lj) / (Lj+1 + Lj)

Lj-1 = Lj · (1 − f·mt) / (1 + f·mt)

其中包含一个JND分数f——该参数可调整以匹配各种亮度范围。例如，f = 0.9 时覆盖约10-3到104 cd/m²的动态范围。通过迭代方法构建EOTF后，可直接纳入查找表（LUT）。

图8-10：PQ曲线的推导过程——从CSF到JND亮度映射

为更好地拟合迭代JND表，采用了改进版Naka-Rushton锥体响应方程作为PQ的数学形式：

4.2 PQ 的验证

PQ的性能验证主要通过一系列涵盖多种位深的视觉测试进行。

图11：PQ验证测试所用的测试图样

表1：PQ编码效率对比——平坦场图像中PQ显著减少了所需位深

测试从平坦场（Flat Field）图像开始，呈现一致的D65色度。通过对单个量化级别的有意扰动，评估实现视觉上"不可见"差异所需的位数——即量化失真小到人眼无法辨别。

图12：典型视频图像测试素材

表2：在典型视频图像中，PQ编码仍表现出显著的位深效率优势

作为补充策略，PQ性能验证涵盖了典型视频图像——包含广泛的视觉数据和不可预测的动态变化，对PQ构成了更接近真实世界的挑战。结果表明PQ在消除条带伪影和减少视觉退化方面表现优异。

五、结论

CSF在HDR编码中的应用已被证明是优化视觉表示质量的成功策略。这一创新过程将人类视觉感知特性与先进编码技术相结合，提供了无与伦比的视觉成果。

从Weber-Fechner的对数定律，到Stevens的幂律，再到Barten将空间频率纳入考量的CSF模型，最终汇聚为感知量化器（PQ）——这条路径展示了基础视觉科学与工程应用之间深刻的协同关系。随着我们在这一激动人心的研究领域不断取得进展，对CSF在HDR编码中更广泛的理解和更深入的实现有望彻底改变我们与数字视觉内容的关系和交互方式。
